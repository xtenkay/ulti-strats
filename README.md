## Running the Project

If you're planning to make a PR, just clone the project and run it locally.  
It's a simple Astro project.

### Astro CLI Docs

[Astro CLI Reference](https://docs.astro.build/en/reference/cli-reference/?utm_source=chatgpt.com)

### Setup

You’ll probably need to run this too:

```bash
npx astro sync
```
