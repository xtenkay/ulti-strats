## Basic Use

```js
  import Grid from '@/components/mdx/Grid.astro'
```

```mdx
<Grid>
  <p class="p-4 bg-primary text-white">1</p>
  <p class="p-4 bg-primary text-white">2</p>
</Grid>
```

## Example

<Grid>
  <p class="p-4 bg-primary text-white">1</p>
  <p class="p-4 bg-primary text-white">2</p>
</Grid>

---

## Specifying Grid Columns

You can specify the number of columns for the grid by passing the `cols` prop. The value should be a number between 1 and 12.

<Grid cols='3'>
  <p class="p-4 bg-primary text-white">1</p>
  <p class="p-4 bg-primary text-white">2</p>
  <p class="p-4 bg-primary text-white">3</p>
</Grid>

```mdx
<Grid cols='3'>
  <p class="p-4 bg-primary text-white">1</p>
  <p class="p-4 bg-primary text-white">2</p>
  <p class="p-4 bg-primary text-white">3</p>
</Grid>
```

---

## Using with Other Components

You can use other components in the grid as well:

<Grid cols='3'>
  <p class="p-4 bg-primary text-white">1</p>
  <Card title="Card Component">
    <p class="p-4 bg-primary text-white">2</p>
  </Card>
  <p class="p-4 bg-primary text-white">3</p>
</Grid>

```mdx
<Grid cols='3'>
  <p class="p-4 bg-primary text-white">1</p>
  <Card title="Card Component">
    <p class="p-4 bg-primary text-white">2</p>
  </Card>
  <p class="p-4 bg-primary text-white">3</p>
</Grid>
```
