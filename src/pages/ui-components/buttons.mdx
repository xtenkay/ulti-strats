## Using utilities with buttons

### Button size

Wrap the button in a container that uses the [font-size utility classes](/utilities/typography) to scale buttons:

<Button class="btn btn-lg mr-4" href="http://example.com/">Big Button</Button>

<Button class="btn btn-sm" href="http://example.com/">Tiny Button</Button>


```markdown
<Button class="btn btn-lg" href="http://example.com/">Big Button</Button>

<Button class="btn btn-sm" href="http://example.com/">Tiny Button</Button>
```

### Spacing between buttons

Use the [margin utility classes](/utilities/layout/) to add spacing between two buttons in the same block.


<Button class="btn mr-3" href="http://example.com/">Button with space</Button>
<Button class="btn" href="http://example.com/">Button</Button>

***

<Button class="btn mr-10" href="http://example.com/">Button with more space</Button>
<Button class="btn" href="http://example.com/">Button</Button>


```markdown
<Button class="btn  mr-3" href="http://example.com/">Button with space</Button>
<Button class="btn" href="http://example.com/">Button</Button>

<Button class="btn mr-10" href="http://example.com/">Button with more space</Button>
<Button class="btn" href="http://example.com/">Button</Button>
```
