## Font stack

By default, Astro Just Docs uses a native system font stack for sans-serif fonts:

```scss
system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
```

<p class="tracking-widest px-4">ABCDEFGHIJKLMNOPQRSTUVWXYZ</p>
<p class="tracking-widest px-4">abcdefghijklmnopqrstuvwxyz</p>

For monospace type, like code snippets or the `<pre>` element, Astro Just Docs uses a native system font stack for monospace fonts:

```scss
"SFMono-Regular", Menlo, Consolas, Monospace
```

<p class="tracking-widest px-4">ABCDEFGHIJKLMNOPQRSTUVWXYZ</p>
<p class="tracking-widest px-4">abcdefghijklmnopqrstuvwxyz</p>

## Responsive type scale

Astro Just Docs uses a responsive type scale that shifts depending on the viewport size.

| Selector              | Small screen size `font-size`    | Large screen size `font-size` |
|:----------------------|:---------------------------------|:------------------------------|
| `h1`, `.h1`           | 32px                             | 36px                          |
| `h2`, `.h2`           | 18px                             | 24px                          |
| `h3`, `.h3`           | 16px                             | 18px                          |
| `h4`, `.h4`           | 14px                             | 16px                          |
| `h5`, `.h5`           | 16px                             | 18px                          |
| `h6`, `.h6`           | 18px                             | 24px                          |
| `body`                | 14px                             | 16px                          |

## Headings

Headings are rendered like this:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

`.h1`, `.h2`, `.h3`, `.h4`, `.h5`, `.h6` Headding elements will automtaically be styled appropratiately, but classes can also be given to any element to apply the corresponding heading style.
